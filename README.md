## ✨ Features

- 🔺 **Interactive 5-level food pyramid** built with CSS clip-path
- 👆 **Clickable levels** — each opens a detailed modal with full info
- 📋 **Rich modal panels** — description, benefits, examples, frequency, nutrition meter
- 🎓 **Student cards section** — each student introduces their topic
- 💡 **Quick Nutrition Tips** section
- 🌙 **Dark mode toggle** with localStorage persistence
- 📱 **Fully responsive** — desktop, tablet, and mobile
- 🎞️ **Smooth animations** — scroll reveals, staggered entrance, hover effects
- ♿ **Accessible** — keyboard navigation, ARIA roles, focus trap
- ⚡ **Zero dependencies** — pure HTML, CSS, JavaScript

## 🚀 How to Run Locally

### Option A — VS Code + Live Server (Recommended)

1. Install [Visual Studio Code](https://code.visualstudio.com/)
2. Install the **Live Server** extension (by Ritwick Dey)
3. Open the `healthy-food-pyramid/` folder in VS Code
4. Right-click `index.html` → **"Open with Live Server"**
5. Opens at `http://127.0.0.1:5500/index.html` ✅

### Option B — Direct browser

Double-click `index.html` in your file explorer.  
Works in Chrome, Edge, and Firefox without any server.

> **Note:** Google Fonts are loaded from the internet on first use, then cached. The site works fully offline after the first load.

## 🌐 How to Deploy on GitHub Pages

### Step 1 — Create a GitHub repository
1. Go to [github.com](https://github.com) and sign in (or create a free account)
2. Click **"New"** (green button) → name it `healthy-food-pyramid`
3. Set it to **Public** → click **"Create repository"**

### Step 2 — Upload the files
1. On the repository page, click **"uploading an existing file"**
2. Drag and drop: `index.html`, `style.css`, `script.js`, `README.md`
3. Click **"Commit changes"**

### Step 3 — Enable GitHub Pages
1. Go to **Settings** tab of your repository
2. Scroll to **"Pages"** in the left sidebar
3. Under **"Source"** → select **"Deploy from a branch"**
4. Under **"Branch"** → select **"main"** → click **Save**

### Step 4 — Get your link
Wait 1–2 minutes, then your site will be live at:
```
https://YOUR-USERNAME.github.io/healthy-food-pyramid
```
